checkIfPlayerWon

Run through all the possible winning combinations to see if a single set of winning combinations is contained in the players squares array. If so, return true for the ternary operator in playTurn, and set game.winner to the current player.

endCurrentGame

Set the game state when game is won. Disable event listeners on elements used for game. Creates a winner overlay and appends the winner, and a button to play again. If no game winner is assigned, when 9 moves have been played it announces a tie.

playTurn

Adds color to targetted square, sets aria-attributes for accessibility. Records the value and color in game.moves to keep a record. Adds the target value to the appropriate player squares field. Calls CheckIfPlayerWon to check for win, or if there have been 9 moves, game must be tied. If won or tied, call EndCurrentGame to set end game state. Otherwise, hand the turn over to the other player.
